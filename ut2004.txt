Unreal Tournament 2004
 - download from steam
 - download d3d8to9 (https://github.com/crosire/d3d8to9)
 - put d3d8.dll file into "C:\Program Files (x86)\Steam\steamapps\common\Unreal Tournament 2004\System" directory

64bit build uses newer d3d9 render, but it's missing graphical  effects compared to the d3d8 render. be sure to use d3d8to9 wrapper
