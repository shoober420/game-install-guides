- download MSYS2 (https://www.msys2.org/)

- pacman -S mingw-w64-x86_64-SDL2 make mingw-w64-x86_64-gcc python vim
- download Super Metroid (JU) ROM (JU = Japan USA)
- rename ROM file "sm.smc"
- put in "sm" directory
