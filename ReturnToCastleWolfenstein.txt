Return to Castle Wolfenstein (DONT USE REALRTCW MOD OR VENOM MOD, NON-VANILLA / CHANGES MODELS, SOUNDS, TEXTURES ETC)

- download from steam
- download iortcw (https://github.com/iortcw/iortcw)
- download latest OpenAL (https://github.com/kcat/openal-soft)



- delete old bundled OpenAL64.dll from iortcw
- rename soft_oal.dll from latest downloaded OpenAL to OpenAL64.dll
- put renamed OpenAL64.dll into RTCW directory
