Paper Mario 64

- open MSYS2 MINGW64
- git clone https://github.com/pmret/papermario
- pacman -S mingw-w64-x86_64-libelf
- pacman -S mingw-w64-i686-libelf
- pacman -S mingw-w64-clang-x86_64-libelf
- pacman -S mingw-w64-clang-i686-libelf
- pacman -S mingw-w64-x86_64-autotools
- pacman -S mingw-w64-i686-autotools
- edit install_deps.sh to only include Arch Linux portion of code
- also remove the uname code from Arch Linux section of install_deps.sh 
- ./install_deps.sh
- ./install_compilers.sh
