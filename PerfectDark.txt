Perfect Dark

- download MSYS2 (https://www.msys2.org/)
- install MSYS2 in "C:\msys64" directory
- launch mingw64.exe in msys2 directory
- pacman -S mingw-w64-i686-toolchain mingw-w64-i686-SDL2 mingw-w64-i686-zlib make git
- git clone --recursive https://github.com/fgsfdsfgs/perfect_dark.git


- download perfectdark port (https://github.com/fgsfdsfgs/perfect_dark)
- download perfect dark ROM, USA V1.1 NTSC-FINAL recommended
- put ROM in "pd-i686-windows\data" directory
- name ROM "pd.ntsc-final.z64"
