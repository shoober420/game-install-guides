POD Gold
 - download pod gold from gog
 - install pod gold (GOG_Galaxy_POD_Gold.exe)
 - download podhacks (https://svn.nicode.net/podhacks/bin/) (PodHacks.zip)
 - install podhacks.exe into "C:\Program Files (x86)\GOG Galaxy\Games\POD GOLD"
 - download nGLide (https://www.zeus-software.com/downloads/nglide)
 - go to "C:\Windows\SysWOW64" and put glide.dll, glide2x.gll, glide3x.dll, 3DfxSpl.dll, 3DfxSpl2.dll, 3DfxSpl3.dll, nglide_config.exe, nglide_readme.txt in "C:\Program Files (x86)\GOG Galaxy\Games\POD GOLD"
 - run nglide_config.exe and setup
 - download original retail engine acceleration sound file (https://drive.google.com/file/d/1gdYSvhVggaZPof8nq4fQF6_gogXm-LaR/view)
 - put FIX.MEG in "C:\Program Files (x86)\GOG Galaxy\Games\POD GOLD\DATA\BINARY\RESSON" directory

Fix game running too fast
 - open glide_config.exe
 - Video backend: DirectX (Vulkan too fast)
 - Refresh rate: 60Hz
 - Vertical Synchronization: On (default)
