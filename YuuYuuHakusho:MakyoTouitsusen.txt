Yuu Yuu Hakusho: Makyo Touitsusen

- BlastEm Nightly Builds (https://www.retrodev.com/blastem/nightlies)

WINDOWS ONLY
- download chocolately (InstallChocolately.bat)
- download mercurial (choco install hg)
 + NOT RECOMMENDED, installs random Windows Updates

BlastEm
 - download source manually (https://www.retrodev.com/repos/blastem/archive/tip.zip)
 - hg clone https://www.retrodev.com/repos/blastem (prefer to use mercurial in MSYS2 instead)

- download Yuu Yuu Hakusho: Makyo Touitsusen jap rom
- download fan transtion (https://www.romhacking.net/translations/1222/)
- download Lunar IPS (https://www.romhacking.net/utilities/240/)
- extract translation and lunar ips program
- open Lunar IPS
- click "Apply IPS Patch"
- select Yu Yu Hakusho Jap-Eng Translation.ips
- select Yuu Yuu Hakusho: Makyo Touitsusen jap rom
- patch and launch BlastEm

DONT PRESS THE FOLLOWING KEYS DURING GAMEPLAY:
- C, V, B, N, and U
- Triggers error: Failed to lock texture: LockRect(): INVALIDCALL
- Caused by Debugger keybinds
